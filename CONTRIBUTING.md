# Panduan Kontribusi (Contributing)

Terima kasih telah tertarik untuk berkontribusi pada proyek ini! Kami sangat menghargai bantuan Anda.

## Jenis Kontribusi
Semua kontribusi sangat diterima, termasuk namun tidak terbatas pada:
* **Laporan Bug**: Memberitahu kami jika ada sesuatu yang tidak berfungsi.
* **Perbaikan Bug**: Mengajukan pull request untuk memperbaiki masalah yang ada.
* **Peningkatan Dokumentasi**: Memperbaiki typo atau menambah penjelasan di README.
* **Peningkatan Fitur**: Menambah fungsionalitas baru pada script yang ada.
* **Ide-Ide**: Memberikan saran atau masukan yang membangun untuk kemajuan proyek ini.

## Cara Berkontribusi
1. Fork repositori ini.
2. Buat branch baru untuk fitur atau perbaikan Anda.
3. Commit perubahan Anda dengan pesan yang jelas.
4. Push ke branch Anda dan buat Pull Request.

Kami menantikan kontribusi luar biasa dari Anda!
